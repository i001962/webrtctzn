# WebRTCTZN

Decentralized p2p dWebRTC for CTZN with multi-network peer discovery.

## What does this do?
Audio & Video Chat, Whiteboarding, Image and Screensharing with a minimal interface.

[Try It Now](https://webrtctzn.glitch.me)
